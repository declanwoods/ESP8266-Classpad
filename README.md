# ESP8266-Classpad

### Introduction

ESP8266-Classpad is a project that aims to allow two or more Casio Classpads to communicate wirelessly (through HTTP) via the 2.5mm communication port.

### Usage

To run the node.js game server:
```

cd ESP8266-Classpad
npm install
node server.js
```
